Build Steps
-----------

### Windows

1) Install [Visual Studio Build Tools][1].
2) Open a Developer Command Pprompt.

                | Visual Studio 2017 
                            | Visual Studio Tools 
                                        | Developer Command Prompt for VS 2017
                                        
3) `cd wungchungery & nmake /f Makefile.Win32`
4) Output executables should be in builds directory.
